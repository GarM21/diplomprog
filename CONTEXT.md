# Проект дипломной работы

Тема:
Разработка программного обеспечения определения площади зелёных насаждений территории по спутниковым снимкам.

Требования:
- desktop приложение на Python
- GUI на PyQt5
- анализ спутниковых снимков
- определение процента зелёных насаждений
- использование обучаемых алгоритмов
- модель сегментации U-Net
- OpenCV + TensorFlow/PyTorch

Функции:
- загрузка снимка
- анализ изображения
- построение маски зелёных зон
- вычисление процента озеленения
- отображение результата в GUI
